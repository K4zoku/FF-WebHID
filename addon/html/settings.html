WebHID Settings

Daemon as NM host

Talk to the daemon directly over native messaging (skip the webhid-native-messaging forwarder and Unix socket). Requires the webhid-daemon-nm-host native-messaging manifest to be installed and the daemon to run as your user (with the udev rule). Lower latency, but no root-daemon fallback. The addon reconnects automatically when toggled.

Fire-and-forget sendReport

Resolve sendReport Promise immediately without waiting for daemon ack (faster, but no error reporting on write failure)

SharedArrayBuffer data plane

Use WebSocket + SharedArrayBuffer for high-performance input/output reports. Injects COOP/COEP headers which may break some sites. Disable for maximum site compatibility (falls back to slower postMessage path)

SAB Buffer Capacity (default)

Ring-buffer slots for input reports. Higher = fewer drops under heavy traffic. Per-site overrides in popup.

Log Level

Controls console output verbosity. Error = errors only, Warn = +warnings, Info = +info, Debug = +debug (includes performance timing). Reload pages after changing.

Performance Timing

Log timing for sendReport, input reports, and WebSocket frames. Only effective at Debug log level.
