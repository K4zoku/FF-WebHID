FF-WebHID Settings

Performance Logging

Log [webhid-timing] and [worker] timing messages to page/addon console

Fire-and-forget sendReport

Resolve sendReport Promise immediately without waiting for daemon ack (faster, but no error reporting on write failure)

SharedArrayBuffer data plane

Use WebSocket + SharedArrayBuffer for high-performance input/output reports. Injects COOP/COEP headers which may break some sites. Disable for maximum site compatibility (falls back to slower postMessage path)

SAB Buffer Capacity (default)

Ring-buffer slots for input reports. Higher = fewer drops under heavy traffic. Per-site overrides in popup.

Log Level

Controls console output verbosity. Error = errors only, Warn = +warnings, Info = +info, Debug = +debug. Reload pages after changing.
